## 9. СТЕК — ФИНАЛ

**Astro 5 + Vue 3 (islands) + Django REST + PostgreSQL**

Цель: максимальная скорость фронта при простой поддержке бэка. Один бэкенд, одна админка, никаких микросервисов.

### 9.1 Архитектура

```
Контент-менеджер → Django Admin (RU/EN/DE через django-modeltranslation)
                        ↓
                    PostgreSQL 16 + Redis (кеш)
                        ↓
                    DRF API
                        ↓
                  Astro билдит статику (90% страниц) + Vue islands (фильтры, формы)
                        ↓
                    Cloudflare Pages (edge CDN, бесплатно)
```

### 9.2 Полный стек

| Слой | Технология | Зачем |
|---|---|---|
| **Frontend framework** | **Astro 5** | Статика по умолчанию → Lighthouse 95-100, ноль JS на контентных страницах |
| **UI компоненты (islands)** | **Vue 3** | Команда знает Vue, Astro нативно поддерживает Vue 3 |
| **Стили** | **Tailwind 4** | Скорость разработки, маленький output после purge |
| **Анимации** | **Motion One** | ~6kb, GPU-accelerated, замена Framer Motion |
| **Backend** | **Django 5 + DRF** | Сильная админка, знакомо команде по Arent All |
| **i18n** | **django-modeltranslation** + Astro i18n | Переводы прямо в Django Admin, статика по языкам |
| **Импорт продуктов** | **django-import-export** | Загрузка каталога из Excel |
| **DB** | **PostgreSQL 16** | Стандарт |
| **Cache** | **Redis** | Сессии, кеш API-ответов, очереди |
| **Поиск** | **Meilisearch** | Бесплатный, быстрее Algolia, self-hosted |
| **Медиа** | **Cloudflare R2** | S3-совместимое, 10GB бесплатно, без egress |
| **Хостинг front** | **Cloudflare Pages** | Edge CDN, бесплатно для статики, авто-деплой из git |
| **Хостинг back** | **Hetzner Cloud** | €5-10/мес, Германия (по близости к юзерам RU/EN/DE) |
| **Email** | **Mailgun** (free tier 100/день) | Транзакционные письма |
| **Чат** | **Tawk.to** (free) | Без лимитов сообщений |
| **Заявки** | **Telegram Bot API** | Вместо CRM, бесплатно |
| **Аналитика** | **GA4 + Yandex Metrika** | Бесплатно, для RU нужна Метрика |
| **Captcha** | **hCaptcha** | Бесплатно, лучше privacy чем reCAPTCHA |

### 9.3 Что и где рендерится

| Страница | Рендеринг | JS на клиенте |
|---|---|---|
| Главная | Static (build-time) | Только Vue island для слайдера и формы |
| /about/* | Static | 0 KB |
| /news/ | Static (генерируется при публикации) | 0 KB |
| /news/{slug}/ | Static | 0 KB |
| /products/ (каталог с фильтрами) | SSR + Vue island | Vue фильтры (~30 KB) |
| /product/{slug}/ | Static | Только галерея (~10 KB) |
| /faq/ | Static | Минимум для аккордеона |
| /contact/ | Static + Vue island | Только форма (~15 KB) |
| /pds-sds/ | SSR | Vue таблица с фильтрами |

**Итого:** на 80% страниц 0 KB JS. На остальных — только нужный код, не весь рантайм фреймворка.

### 9.4 Целевые метрики

- **Lighthouse Mobile:** 95+ (Performance / SEO / Best Practices / Accessibility)
- **LCP:** < 1.2s
- **CLS:** < 0.05
- **INP:** < 200ms
- **Bundle size:** 0 KB на 80% страниц, < 50 KB на интерактивных
- **TTFB:** < 100ms (CDN edge)

### 9.5 Что НЕ берём и почему

- ❌ **FastAPI** (рассматривалось) — на каталог-сайте без websockets/ML/streaming это лишний сервис без выгоды
- ❌ **Nuxt 3** — Vue с SSR хорош, но на контентном сайте Astro быстрее (статика > SSR)
- ❌ **Next.js** — React, тяжелее Astro, команда любит Vue
- ❌ **WordPress** — медленно на 150+ продуктах, дыра в безопасности
- ❌ **Webflow** — $99/мес × 3 языка, нет нормального бэка, лимиты коллекций
- ❌ **Strapi/Directus** — лишнее звено, Django Admin сильнее
